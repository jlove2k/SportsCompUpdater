# Roadmap

## Sprint 1: Core Updater

- Read workbook.
- Extract player names from column A.
- Combine regular season and playoff totals.
- Update columns E through P.
- Preserve formatting.
- Save an updated workbook.
- Produce unmatched-player reports.

## Sprint 2: Desktop GUI

- Add a file picker.
- Add season selection.
- Show matched and unmatched players before saving.

## Sprint 3: Better Data Providers

- Add an ESPN-backed provider if a stable source is available.
- Keep `nba_api` as a reliable fallback.

## Sprint 4: Packaging

- Package as a macOS app.
- Add a signed release artifact.
